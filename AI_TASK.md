# AI Task

Обнови правильно сделай  полноценное приложение на Python + GitHub Actions:

Задача: поисковый робот, который каждый день заходит на сайт https://tv.l2arena.fun , находит ВСЕ рабочие m3u8-ссылки на каналы (включая ссылки, спрятанные в playerjs.html?file=...), и формирует из них M3U-плейлист.

Требования:
1. Парсер должен уметь обрабатывать динамический контент (JavaScript). Использовать Selenium, Playwright или requests+анализ сетевых запросов.
2. Извлекать не только прямые .m3u8, но и ссылки из параметра file= в URL типа /playerjs.html?file=https://...
3. Каждой найденной ссылке давать человеческое название канала (можно вытащить из текста на странице или из имени файла).
4. Формировать файл playlist.m3u с заголовком #EXTM3U и записями:
   #EXTINF:-1,Название канала
   https://....m3u8
5. GitHub Actions должен запускать скрипт каждый день (cron: 0 6 * * *) и автоматически коммитить/пушить обновлённый playlist.m3u в репозиторий.
6. В репозитории должны быть: scraper.py, requirements.txt, .github/workflows/update.yml, README.md с инструкцией по запуску и использованию.
7. Код должен быть чистым, с обработкой ошибок, логированием.

Напиши полный код всех файлов. Никаких "примеров" — всё готовое к работе. с README БЛАТНОМ
